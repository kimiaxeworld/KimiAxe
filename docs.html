Quick Start
Get started in minutes
1. Install SDK
npm install kimiaxe-sdk
2. Initialize
const KimiAxe = require("kimiaxe-sdk");
const client = new KimiAxe({
  apiKey: "your_api_key_here"
});
3. Send SMS
await client.sms.send({
  to: "+919876543210",
  message: "Hello from KimiAxe!",
  sender_id: "KIMIAX"
});
4. Shorten Link
const link = await client.links.shorten({
  url: "https://yoursite.com/very-long-url",
  alias: "my-link",
  expiry: "30d"
});
Authentication
API Keys

All API requests require authentication using your API key in the Authorization header.

curl -X POST https://api.kimiaxe.com/v1/sms/send   -H "Authorization: Bearer YOUR_API_KEY"   -H "Content-Type: application/json"   -d '{"to":"+919876543210","message":"Hello!"}'
